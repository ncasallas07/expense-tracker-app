# ExpenseTrackerApp

App Android en **Kotlin** para mantener un control al registrar ingresos/gastos, ver saldo y estadísticas (almacenamiento en memoria) con el fin de llevar unas finanzas estables al saber en que se van los gastos.


# Autores
Nelson Casallas - Alejandra Mesa

## Funcionalidades
- Alta de movimientos (fecha automática, categoría, monto, tipo).
- Lista con filtros por **mes** y **categoría**.
- **Saldo** en tiempo real.
- **Gráfico** de torta Ingresos vs Gastos.

## Tech
Kotlin, Android SDK, ViewBinding, MPAndroidChart.

## Ejecución
1. Abrir en **Android Studio** (File → Open).
2. Verificar `jitpack.io` en `settings.gradle` (ya incluido).
3. Seleccionar emulador o dispositivo físico.
4. Presionar ▶️ **Run 'app'**.
